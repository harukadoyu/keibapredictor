                        ************************
                        　　調教分析データ仕様
                        ************************
                                第２版e

概要            ：中間の調教過程を分析したデータです。

ファイル名      ："CYB" + 年月日 + ".txt"
                  例）2008年9月13日の場合： CYB080913.txt
                  (サーバ上は lha にて圧縮されています。拡張子は ".lzh")

更新日時        ：金曜日 19:00頃
                  土曜日 19:00頃

ファイル形式    ：固定長テキストファイル

                                        OCC      ：繰り返し定義
                                        BYTE     ：項目長
                                        TYPE    9：数字 (0のとき0)
                                                Z：数字（0のとき空白）
                                                X：文字
                                                F：16進数(数字 or 小文字アルファベット)
                                                
                                        ※数字は、"-" マイナス記号有り
*******************************************************************************
項目名          OCC     BYTE    TYPE    相対    備考    レコード長：96BYTE
*******************************************************************************

※1 コード等の詳細説明は、【調教分析データ内容の説明】を参照してください。

※2 特定のレース／競走馬でスタッフの調教に対するコメントが入ることがあります。
    コメントがない場合は”全角スペース詰め”です。

※3　既知の不具合
「追切指数」の３byte　ZZ9　が仕様的には、右詰めが正ですが、左詰めで入っています。
既存プログラムを考慮し、修正は次回、版数変更（CYC）の時に致します。
